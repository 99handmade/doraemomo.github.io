Posted on 2020-04-06 Edited on 2020-10-14 Disqus:

keytool

Check APK signature information

keytool -printcert -jarfile [apk_file_name]

Check keystore file information

keytool -list -alias [alias_name] -keystore [keysotre_file_name]

This needs to entrer password.

Sign an apk

jarsigner -verbose -keystore [keysotre_file_name] [apk_file_name] [alias_name]

Verify a signed apk

jarsigner -verify [apk_file_name]
