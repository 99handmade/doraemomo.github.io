Objective C

static variable

Static variables can be used to declare class property similar to Java, which feature that ObjC do not have itself. The declaration will be out of the scope of the class (@implementation), but it must be in the class file. During the initialization of the class object, this property will be initialized only once. This technic is also used to create singleton instances.
